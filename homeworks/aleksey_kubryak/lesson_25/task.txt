Задача 1.

Предположим, вы разрабатываете интернет-магазин и вам нужно создать систему корзины для покупателей. Ваша задача состоит в том, 
чтобы разработать классы для товаров и корзины, а также написать код для добавления товаров в корзину, удаления товаров из корзины, 
обновления количества товаров в корзине и подсчета общей стоимости заказа. 

Класс "Товар" должен содержать такие свойства, как наименование, цена, описание и количество. Он также должен иметь методы для 
получения и установки значений этих свойств.

Класс "Корзина" должен содержать массив объектов класса "Товар", а также методы для добавления, удаления и обновления количества товаров 
в корзине. Он также должен иметь метод для подсчета общей стоимости товаров в корзине.

Задача 2.

Система управления библиотекой. Для этого нам потребуются три класса: Book, Author и Library.

Класс Book будет иметь следующие свойства:
• title
• author_id
• is_available

Класс Author будет иметь следующие свойства:
• id
• last_name

Класс Library будет иметь следующие свойства:
• Books

Теперь нам необходимо реализовать методы в классе Library. Например:
• addBook(Book $book) - добавляет книгу в библиотеку
• removeBook(Book $book) - удаляет книгу из библиотеки
• getBooks() - возвращает все книги из библиотеки
• getAvailableBooks() - возвращает все доступные книги из библиотеки
• getBookByTitle($title) - возвращает книгу по ее названию
• getBooksByAuthor(Author $author) - возвращает все книги автора
