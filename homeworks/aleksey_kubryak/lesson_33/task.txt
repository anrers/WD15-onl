1. Проверка сбалансированности скобок. Напишите функцию на PHP, которая принимает строку, 
содержащую скобки (круглые (), квадратные [] и фигурные {}), и проверяет, является ли эта 
строка сбалансированной в отношении скобок. Функция должна возвращать true, если скобки
сбалансированы, и false в противном случае.

Пример использования:

$string1 = "(([{()}]))";
$result1 = isBalanced($string1);
echo $result1; // Вывод: true
$string2 = "({[)}]";
$result2 = isBalanced($string2);
echo $result2; // Вывод: false

$number = 12345;
